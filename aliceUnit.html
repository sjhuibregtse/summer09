Alice Programming

While being introduced to the world of Computer Science we will use Alice, an easy-to-use system for programming in a 3-D virtual world, to create numerous movie-style animations. Through Alice students will learn how to use various object-oriented programming concepts such as objects, classes, methods, loops, functions, and many more.
