Programming Web Pages with HTML

We will also learn how to create a web page using HTML that can be shared with family and friends. Students will be exposed to basic HTML programming tags and attributes, along with CARP (Contrast, Alignment, Repetition, and Proximity) web design principles.

Day 1 - Vocabulary
  • HyperText Markup Language (HTML)
  • tag
  • required tags
  • angled brackets
  • forward slash "/"
  • title bar
  • text editor (Notepad)
  • browser (Internet Explorer)
  • empty tag
Day 1 - Tags
  • Required Tags:
    • <html>...</html>
    • <head>...</head>
    • <body>...</body>
    • <title>...</title>
  • Paragraph Tag: <p>...</p>
  • Break Tag: <br />
  • Heading Tags:
    • <h1>...</h1>
    • <h2>...</h2>
    • <h3>...</h3>
    • <h4>...</h4>
    • <h5>...</h5>
    • <h6>...</h6>

Day 2 - Tags & Attributes
  • Horizontal Rule: <hr />
  • Horizontal Rule Attributes:
    • color: <hr color="red" />
    • width (percentage):
         <hr width="50%" />
    • width (pixel):
         <hr width= "50" />
    • size (pixel ONLY):
         <hr size="10" />
    • align:
      • left:
          <hr align="left" />
      • right:
          <hr align="right" />
      • center (default):
           <hr align="center" />

  • unordered list:
    • <ul>
        <li>text/content>
      </ul>

  • ordered list:

      </ol>
      • type="1" - Default
      • type="a"
      • type="A"
      • type="i"
      • type="I"
  • font tag & attributes:
    • <font>text to change</font>
      • color="red"
      • face="Tahoma"
